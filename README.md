<h1> Bamazon </h1>

<h2> Logic </h2>
<ul>
  <li> An interactive shopping application that allows the user to choose an item from the list of products by the "item ID"</li>
  <li> Once the user chooses which item to purchase, the application that asks how many of the item that they would like to buy</li>
  <li> After the user has chosen the item they would like to purchase and the amount they would like to buy, the application then returns the total amount of money owed </li>
  </ul>
